Searchable input with jQueryUI

You selected !

User input to allow search and select of set, hardcoded values. In this case, it's states. Code forces selection from list only. Input not found will return "No results."

Embed tag

Jump to RequireJS module

<div style="clear:both;">
	<input id="search-box" type="text" placeholder="Search for a county..."/>
	<div id="no-results" style="display:none;"></div>
	<div id="search-results" style="display:none;">
		<p>You selected <span id="selection"></span>!</p>
	</div>
</div>
